Incident trail

Math integrity bugs — fixed before current data

Wilson float residue, ranking self-comparison, and zero-hidden-weight functional passes all received direct regression tests.

Clean-room auto-exec injection — path layer fixed

conftest/sitecustomize/pytest config and .pth variants are blocked, including mixed case and symlink capture. Host/runtime isolation remains open.

Ollama parser and connection failures — partly fixed

In-block FILE markers are parsed and explicit transport failures are voided. Other response-format failures remain opaque no-edits.

Background jobs reaped — resumability added

Per-run SQLite persistence prevents total progress loss, but resume resets the model call counter and therefore seed schedule.

Undisclosed gap-fill — fully removed

All five model rows are now genuine DB data. Oracle/noop are explicit synthetic baselines. The legacy 500-run forensic gap remains.

What must happen next

  1. Repair the benchmark contract. Quarantine and fix expression-evaluator and redirect validation; derive adversarial tests from the descriptions; rerun affected cells.
  2. Choose one executable scoring specification. Reconcile macro/micro ranking, timeout/infra taxonomy, deterministic evidence, Q-unavailable semantics, and continuous scoring; version it.
  3. Build a real trust boundary. Run agents and graders in pinned, network-disabled isolation and pass agents a sanitized task view without host reference/hidden paths.
  4. Rerun the complete current pack. Derive stable seeds from evaluation/model/task/run identity and persist patches, tests, transcripts, parser outcomes, digests, config, commands, and timings.
  5. Only then build the product surface. Add DB constraints/migrations, raw-run drilldown, diff/test/transcript views, and CSV/JSON export before a dashboard polish sprint.
Do not do yet:
  • Add more models to the contaminated/stale matrix.
  • Call Wilson rank ranges statistically proven ties.
  • Build a glossy dashboard over missing forensic data.
  • Claim untrusted-agent safety, universal model ability, or a fair current-pack leaderboard.
